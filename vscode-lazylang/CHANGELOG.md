# Change Log

All notable changes to the "lazylang" extension will be documented in this file.

## [0.2.0] - 2024-11-11

### Added
- Document formatting support via `lazy format` command
- Automatic formatter discovery from workspace or PATH
- Style guide documentation for Lazylang code formatting

## [0.1.0] - 2024-11-11

### Added
- Initial release
- Syntax highlighting via LSP semantic tokens
- TextMate grammar for basic syntax highlighting
- Language configuration (brackets, auto-closing pairs, comments)
- LSP server integration with automatic discovery
- Configuration options for LSP server path and tracing
